# Skill proof pack template

Use this template when graduating a skill from "useful playbook" to "trusted Titan asset."

Required files:

- `BENCHMARK.md`: reproducible baseline-vs-Titan protocol.
- `RUBRIC.md`: scoring criteria and weights.
- `CASE_STUDY.md`: safe demonstration with inputs, output, and lessons.
- `FAILURE_PATTERNS.md`: known misses and anti-patterns.
- `EVALUATION.md`: scored results, confidence, uncertainty, limitations.
- `EXAMPLES.md`: beginner, startup, enterprise, failure, recovery, and real-world examples.

Do not add these files inside a portable Codex skill folder unless the target runtime expects them. Keep the runtime skill lean; keep proof artifacts in `benchmarks/`, `case-studies/`, `evals/`, or documentation.
